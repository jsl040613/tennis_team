## 🧩 技术栈

- 后端框架：Flask
- 数据库：MongoDB Atlas
- 前端模板：HTML + CSS（Jinja2 渲染）
- 图片上传：Flask 表单上传
- 项目运行环境：Python 3.x



## ✨ 功能一览

- ✅ 用户注册 / 登录 / 认证
- ✅ 球场预定
- ✅ 发布互动动态（文字、图片）
- ✅ 点赞、评论互动
- ✅ 用户个人主页查看发布记录
- ✅ 粉丝管理
- ✅ 我的关注
- ✅ 响应式布局优化（适配不同设备）


##  安装依赖
pip install -r requirements.txt

##  创建并激活虚拟环境
# Windows
python -m venv venv
venv\Scripts\activate

##  运行项目
python app.py
访问项目主页：http://127.0.0.1:5000
